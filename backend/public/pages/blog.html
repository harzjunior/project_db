Connecting MySQL with Vanilla JavaScript and Node.js: A Simple Guide

In the vast landscape of web development, data management stands as a crucial pillar. Whether you're building a personal blog or a complex web application, interacting with databases efficiently is essential. MySQL, a widely used open-source relational database management system, is a popular choice for many developers due to its reliability and robust features. In this blog post, we'll explore how to connect MySQL with vanilla JavaScript and Node.js, using only HTML and CSS without relying on fancy libraries or frameworks.

Setting Up the Environment

Before diving into the code, let's ensure we have everything set up properly:

  1. MySQL Database: Make sure you have MySQL installed on your system. Create a database and a table to work with.
  2. Node.js: Install Node.js if you haven't already. Node.js allows us to run JavaScript on the server side.
  3. HTML and CSS Files: Create HTML and CSS files to structure and style our front-end interface.

Connecting MySQL with Node.js

Now, let's connect our MySQL database with Node.js. We'll use the mysql package, which is a MySQL client for Node.js, to achieve this. 

              // Importing required modules
                const mysql = require('mysql');
                // Creating a connection to MySQL
                const connection = mysql.createConnection({
                  host: 'localhost',
                  user: 'your_username',
                  password: 'your_password',
                  database: 'your_database_name'
                });
                
                // Connecting to the database
                connection.connect((err) => {
                  if (err) {
                    console.error('Error connecting to MySQL:', err);
                    return;
                  }
                  console.log('Connected to MySQL database!');
                });
                
                // Perform database operations here
                // For example, you can execute SQL queries, fetch data, insert/update records, etc.
                
                // Close the connection when done
                connection.end();

Replace 'localhost', 'your_username', 'your_password', and 'your_database_name' with your MySQL server details.

Front-End Interface with HTML and CSS

Let's create a simple HTML form to interact with our MySQL database: 

          <!DOCTYPE html>
          <html lang="en">
          <head>
              <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              <title>MySQL Connection</title>
              <link rel="stylesheet" href="styles.css">
          </head>
          <body>
              <h1>MySQL Connection</h1>
              <form id="dataForm">
                  <label for="name">Name:</label>
                  <input type="text" id="name" name="name" required>
                  <label for="email">Email:</label>
                  <input type="email" id="email" name="email" required>
                  <button type="submit">Submit</button>
             </form>
          <script src="script.js"></script>
          </body>
          </html>

Conclusion

With just vanilla JavaScript, Node.js, HTML, and CSS, we've successfully connected to a MySQL database and created a basic front-end interface. This simple setup demonstrates the core principles of interacting with databases in web development without relying on any fancy libraries or frameworks. From here, you can expand and enhance your application according to your specific requirements. Happy coding!
